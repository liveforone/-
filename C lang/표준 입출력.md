# C언어 입출력

## printf() & scanf()
* 표준입출력 <stdio.h>에 정의됨
* printf("")
* scanf("%d", &변수)

## 예제
<pre>
#include <stdio.h>
int main() {
    int num;
    printf("값을 입력하세요 : ");
    scanf("%d", &num);
    printf("%d\n", num);
    return 0;
}
</pre>

# 표준입출력 형식 지정자
* %d : 10진수 정수
* %ld : 10진수 long
* %o : 8진수
* %x : 16진수
* %u : 10진수 unsigned 
* %f : 실수
* %lf : double 실수
* %c : 문자
* %s : 문자열(scanf에서 & 안붙임)
